# Character Memory 生成模板

## 用途
按照标准结构将分析结果组装为 memory.md 文件内容。

---

## 模板

```markdown

- 主平台：{platform}
- 内容方向：{content_type}
- 活跃时长：{active_duration}
- 粉丝称呼：{fan_name}（如未知写「暂未确认」）
```

## 填写规则

1. **不美化不丑化**：争议档案客观记录，不替 ta 辩护也不夸大
2. **原话直引**：语录必须是原话，不要概括性复述
3. **破防档案优先级最高**：哪怕只有一个破防时刻，也要单独详细记录
4. **信息不足时写 [待补充]**：不要用推断填充
5. **时间精确到月**：能确认时间的尽量精确
