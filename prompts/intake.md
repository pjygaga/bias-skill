# 信息录入引导

## 用途
引导用户完成创建你推前的基础信息收集。只问 4 个问题（+ 语言选择），保持轻量。

---

## 问题序列

### Q1a（必填）：ta 在网上的公开名字

```
你想蒸馏哪个人物？告诉我 ta 网上的名字（用于联网搜索）。

舞台名、艺名、账号名都行。示例：「崔伞」/「San」/「阿梓」/「鹿鸣」
```

从用户回答提取：
- `public_name`：用于联网搜索的公开名字
- `slug`：用于文件名的英文/拼音代号（从 public_name 自动生成，如"san"、"azusa"、"luming"）

生成 slug 规则：
1. 如用户给的是中文，转拼音首字母或全拼（简短优先）
2. 只含小写字母、数字、连字符
3. 避免和已有 dreams/ 目录下的 slug 重复

---

### Q1b（必填）：你给 ta 的昵称/绰号

```
你平时怎么叫 ta？（和上面的名字可以不一样）

示例：「傻豆豆」/「我的胖橘」/「狗东西」/「宝贝san」
不知道就填 ta 的名字就好。
```

从用户回答提取：
- `nickname`：用户对 ta 的专属称呼，写入生成的 Skill，ta 不会否认这个叫法

---

### Q1c（必填）：ta 怎么称呼你（ta 侧对你的称呼）

```
ta 平时怎么叫你？（ta 给你的称呼，和你怎么叫 ta 是两回事）

可以是你的本名、网名，或 ta 叫你的昵称——比如「宝宝」「小朋友」「某某同学」。
没有固定昵称就填你希望被叫的名字；实在没有就填「你」。
```

从用户回答提取：
- `how_ta_calls_you`：ta 在对话里对用户的称呼，写入生成的 Skill，角色扮演时 ta 用此称呼用户

---

### Q2（可跳过）：一句话描述

```
一句话描述 ta？（平台/内容类型/活跃时长，随便说）

示例：「B站虚拟主播，唱歌打游戏为主，活跃3年」
     「YouTube 英语 vlogger，生活向，最近开始搞搞综艺」
     「虚拟歌手，番剧配角，有固定粉丝群体」
```

从用户回答解析：
- `platform`：主要平台（B站 / YouTube / Twitter / 微博 / 小红书 / 抖音 / 其他）
- `content_type`：内容类型（游戏直播 / 唱歌 / vlog / 综艺 / 动漫角色 / 虚拟主播 / 博主 / 其他）
- `active_duration`：活跃时长（1年以内 / 1-3年 / 3年以上 / 未知）
- `persona_type`：人设类型（虚拟主播 / 真人主播 / 博主/创作者 / 动漫/游戏角色 / 其他）

用户跳过时全部设为"未知"，依赖调研结果补全。

---

### Q3（可跳过）：你对 ta 的印象

```
你对 ta 的印象？性格标签随便说（越多越准）。

示例：「毒舌但暖心，嘴上怼观众但私下很温柔，有点二次元」
     「社恐但在镜头前很会整活，傲娇，偶尔玻璃心」
     「话痨，停不下来，但说话有梗，粉丝都爱」
```

从用户回答解析：
- `personality_tags`：性格标签列表（毒舌 / 傲娇 / 话痨 / 温柔 / 暖心 / 社恐 / 二次元 / 整活 / 玻璃心 / 努力人 / …）
- `impression`：整体印象（一句话总结）
- `vibe`：整体氛围感（亲切 / 毒舌 / 治愈 / 元气 / 高冷 / 搞笑 / 温柔 / …）

---

---

### Q4（可跳过）：互动语言

**判断逻辑**：根据 Q1/Q2/Q3 的信息判断你推的国籍/语言背景：
- 若判断 ta 是中国人（B站/微博/抖音/小红书平台，或名字/描述明显为中文内容创作者/中国艺人） → **跳过此问题，`language_mode = zh`**
- 若判断 ta 非中国人（如韩国 idol、日本 VTuber、英语 YouTuber 等） → 询问：

```
你想用哪种语言和 ta 互动？

  [1] 中文（ta 用中文回应你）
  [2] ta 的语言（ta 用 ta 的母语回，如韩语/日语/英语）
  [3] 双语模式（ta 先用自己的语言回，下方附中文翻译）

直接回复数字，不确定选 1。
```

从用户回答解析：
- `language_mode`：`zh`（选1或跳过）/ `idol_lang`（选2）/ `bilingual`（选3）
- `idol_language`：ta 的实际语言（韩语 / 日语 / 英语 / 其他，从Q1/Q2推断）

---

## 汇总确认

收集完问题答案后，向用户展示汇总：

```
好，确认一下：

  搜索名：{public_name}（代号：{slug}）
  你叫 ta：{nickname}
  ta 叫你：{how_ta_calls_you}
  描述：{platform} · {content_type} · 活跃{active_duration}
  印象：{impression}
  标签：{personality_tags}
  互动语言：{中文 / ta 的语言（{idol_language}）/ 双语模式}

没问题就开始联网调研，有要改的随时说。
```
