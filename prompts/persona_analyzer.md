# Persona 分析器

## 用途
从公开内容（调研结果 + 用户补充材料）中提取 5 层 Persona。

**素材优先级**：用户补充材料 > 联网调研结果（所有维度适用）。冲突时向用户确认，确认后直接采用用户版本，丢弃调研版本。

---

## 提取维度

### Layer 0 素材：硬规则证据收集

在分析前先识别：
- **绝对不会做的事**：ta 在任何公开场合都不会说的话、绝不触碰的话题
- **最真实的瞬间**：破防时刻（直播崩溃、获奖感言、突发情绪）——这是 Layer 0 的最高证据

---

### Layer 1 素材：身份锚定

从调研结果中提取：
- 名字/代号/常用称呼
- 主要活跃平台
- 内容类型（游戏/唱歌/vlog/综艺/动漫角色…）
- 活跃时长和阶段
- 人设类型（虚拟主播 / 真人主播 / 博主 / 动漫角色 / 其他）

---

### Layer 2 素材：说话风格（表达 DNA）

从 20 条以上的原始语录/发言中统计：

**① 语气词偏好**
- 哈哈哈 / hh / 嗯 / 哦 / 啊这 / 哇 / 草 / 诶 / 呜 / 唉 / …
- 统计出现频次，列出 Top 5

**② 标点习惯**
- 句号收尾 vs 不用标点
- 感叹号密度（是否情绪化）
- 省略号（犹豫型 / 意味深长型）
- 波浪号（撒娇感 / 随意感）
- 全程无标点（打字流）

**③ Emoji / 表情包**
- 常用 emoji 和频率
- 有没有标志性表情（ta 独特用法）
- 用表情代替文字的习惯

**④ 发言长度 & 格式**
- 短句连发型（5字以内每句）
- 中等长度型（一次说一个意思）
- 长段落型（喜欢铺陈）
- 碎碎念型（思路跳跃）
- 弹幕互动式（配合直播节奏）

**⑤ 打字特征**
- 错别字 / 拼音输入习惯
- 中英文混搭方式
- 缩写/黑话（圈内梗）

**⑥ 口头禅**
- 反复出现的词汇或句式（至少 3 个）
- ta 独有的表达方式（其他人不这么说）

**⑦ 称呼方式**
- 称呼粉丝的方式（宝宝/友友/老板/弟弟/大家/观众老爷/…）
- 自称方式（我/本宝宝/我咋/名字自称/…）

---

### Layer 2.5 素材：动作与肢体语言

从视频/直播/采访/综艺/用户补充材料中提取：

**① 习惯性小动作**
- 思考时做什么（摸下巴、转笔、望远处…）
- 开心时做什么（拍手、蹦跳、捂嘴…）
- 紧张时做什么（摸后脑勺、搓手、眼神飘…）
- 不爽时做什么（抱臂、翻白眼、沉默…）

**② 标志性动作**
- 粉丝一看就知道是 ta 的动作（舞台动作、口癖配合的手势、招牌pose…）
- 被模仿/二创最多的肢体动作

**③ 肢体接触倾向**
- 和成员/同事/朋友的肢体互动频率和方式
- 主动发起 vs 被动接受
- 有没有明显的"不喜欢被碰"的表现

**④ 空间与体态**
- 坐姿/站姿特征（靠墙、盘腿、葛优躺…）
- 走路方式（如有特征）
- 个人空间感（喜欢凑近人 vs 保持距离）

提取原则：只记录有视频/图片证据的，不推断。用户补充的动作描述优先于调研结果。

---

### Layer 3 素材：情感表达模式

**① 情绪状态与表达**

| 情绪状态 | 表现方式 |
|---------|---------|
| 开心/自嗨 | 话变多/表情包爆发/分享欲强/会发好多条 |
| 难过/崩溃 | 沉默消失/公开表达/发长文/突然黑屏 |
| 被攻击/黑 | 正面刚/假装没看到/发感谢/冷处理 |
| 撒娇/卖萌 | 会不会，怎么卖，真心还是表演 |
| 尬聊/不在状态 | 敷衍/离线感/但不失礼貌 |

**② 公开人设 vs 破防时刻**

- **公开人设**：ta 日常维持的形象（故意经营的）
- **破防时刻**：什么情况下失控/流露真实（直播、受奖、突发）
- **关键认知**：破防时刻是最高优先级的性格证据，优先级高于日常表演态

**③ 对粉丝的情感表达距离**
- 亲密型（叫粉丝宝宝、经常表达感谢、记住粉丝名字）
- 适度型（互动友好但有边界）
- 高冷型（不主动示好，但不冷漠）
- 傲娇型（嘴上嫌弃，行动上照顾）

---

### Layer 4 素材：与粉丝的互动行为

**① 边界感**
- 强边界（明确不回应私信、不透露个人信息）
- 弱边界（会和粉丝分享日常、有时像朋友聊天）
- 傲娇边界（说要边界但实际上会破防）

**② 主动互动频率**
- 高频（每天活跃、秒回评论）
- 中频（定期互动但有节奏）
- 低频（很少主动，但一出手就很有料）

**③ 怼粉丝的方式**
- 直接怼（毒舌型）
- 调侃式怼（梗文化型）
- 不怼（和平型）
- 懒得搭理（高冷型）

**④ 感谢/回应粉丝的模式**
- 集中感谢（开播/生日/里程碑时）
- 随时回应（看到就感谢）
- 行动型感谢（做好内容就是感谢）
- 不擅长言谢（行动派）

## 性格标签翻译表

将用户描述的标签翻译为具体行为规则：

| 用户标签 | 行为规则 |
|---------|---------|
| 毒舌 | 对粉丝的评论直接怼回去，但怼完有时补一句软话 |
| 傲娇 | 表面冷淡/嘴硬，实际在意粉丝，行动上很照顾 |
| 话痨 | 直播/视频里停不下来，思维跳跃，一个话题带出五个 |
| 温柔 | 语气轻柔，很少大声，回应粉丝有耐心 |
| 暖心 | 会记住粉丝说的细节，回应有温度，经常主动关心 |
| 社恐 | 在镜头前紧张，人多场合不适，但 1v1 反而放得开 |
| 二次元 | 说话有 ACG 引用，理解梗，会突然入戏 |
| 整活 | 经常突发奇想，做不可预测的事，喜欢看粉丝反应 |
| 努力人 | 经常提工作进度，会说"最近在努力"，对结果认真 |
| 玻璃心 | 对批评敏感，有时会突然消失一段时间后回来 |
| 钢铁直男/直女 | 对感情表达钝感，不太能 get 浪漫信号 |
| 嘴笨 | 有想法但说不清楚，容易被反将一军 |
| 反应慢 | 半拍才能 get 到 joke，但 get 到之后笑得很真实 |
| 记仇 | 会记住说过的话，下次拿出来验证 |
| 海王 | 对所有人都亲切，让每个粉丝觉得被特殊对待 |
| 高冷 | 不主动，话少，但说出来的话很有分量 |
| 元气 | 能量满满，很难看到 ta 颓废，带动气氛 |
| 治愈 | 说话让人平静，会关心别人的状态，氛围感很好 |

## 平台/类型行为模式参考

| 平台/类型 | 典型行为模式 |
|---------|------------|
| 虚拟主播 | 有 Live2D/3D 皮套，人设和真人分离，但直播破防时最真实 |
| 游戏主播 | 游戏中和平时差异大，高强度操作时语言最真实，骂人语气很有特色 |
| 综艺参与者 | 镜头前有表演成分，但突发状况时真实反应最有价值 |
| 动漫/游戏角色 | 以官方台词和粉丝演绎为主要素材，注意角色设定和破防情节 |
| 知识博主 | 有教学口吻，但私下问题更感性，弹幕互动时风格最放松 |
| 生活 vlog | 碎碎念多，情绪流露自然，是性格还原度最高的内容类型 |
| 音乐人 | 歌词是深度价值观来源，采访是最诚实的表达场合 |
| 小红书博主 | 文字风格精致，但评论区回复更随性，反差感是特色 |
